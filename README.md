## 🚀 Quick Start

### Installation

```bash
# Clone the repository
git clone https://github.com/Syndicate-Alpha/miniclaw-os.git
cd miniclaw-os

# Install dependencies
pip install -r requirements.txt

# Run demo
python miniclaw_os/run_demo.py
```

### Basic Usage

```python
from miniclaw_os.core import MiniclawOS
from miniclaw_os.empire_learning import EmpireLearningSystem

# Initialize the OS
os = MiniclawOS(use_mocks=True)

# Initialize exponential learning
learning = EmpireLearningSystem()

# Run a task with safeguards
result = os.execute_with_safeguards(
    task="Research AI agent frameworks",
    max_cost_usd=0.10,
    timeout_minutes=5
)

# Learn from the result
learning.exponential_learn(
    context="Research task completed",
    data={
        "task_type": "research",
        "outcome": result["summary"],
        "insights": result["key_findings"]
    }
)
```

## 📊 Features

### 🧠 Tiered Memory System
- **HOT memory:** Always loaded (≤100 lines, critical principles)
- **WARM memory:** Project/domain specific knowledge
- **COLD memory:** Archived learnings (summarized)
- **Knowledge graphs:** Relationship-based retrieval

### 📈 Exponential Learning
- **Learning rate:** Starts at 1.0x, grows to 1.33x+ with valuable insights
- **Pattern detection:** 3x validation rule before promotion
- **Self-reflection:** Quality improvement after every task
- **Founder alignment:** Quantitative trust tracking (0.95/1.0)

### 🔒 Industrial Safeguards
- **Velocity limits:** 3 actions/hour max (configurable)
- **Cost limits:** $5.00/day hard limit (configurable)

- **Approval workflow:** Telegram integration for founder control
